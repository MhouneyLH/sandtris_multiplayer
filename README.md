# Multiplayer Sandtris

The main motivation behind this fun project is to provide a Sandtris game that can be also played together. Just host the container connect to the game and enjoy!

## Getting Started

1. Clone the repository:

```bash
git clone ...
```

2. Run the Docker containers:

```bash
docker-compose -f docker-compose.dev.yml up --build -d
```

3. Open your web browser and navigate to `http://localhost:8080` to join the queue and start playing!

4. Have fun!
